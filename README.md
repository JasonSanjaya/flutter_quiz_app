#  Quizku

Jason Sanjaya (231401066) Lab 5 

Nama aplikasi: Quizku
Deskripsi aplikasi: 
QuizKu adalah sebuah aplikasi kuis interaktif berbasis Flutter yang dirancang untuk membantu pengguna dalam menguji pengetahuan mereka secara sederhana dan menyenangkan. Aplikasi ini menampilkan serangkaian pertanyaan pilihan ganda yang harus dijawab oleh pengguna. Sebelum memulai kuis, pengguna diminta untuk memasukkan nama mereka, sehingga hasil kuis dapat ditampilkan secara personal.

Selama proses kuis, pertanyaan dan pilihan jawaban ditampilkan satu per satu dengan navigasi Sebelumnya dan Berikutnya. Pengguna dapat melihat progres pengerjaan melalui indikator progres yang menunjukkan posisi pertanyaan saat ini dibandingkan total pertanyaan.

Setelah seluruh pertanyaan terjawab, aplikasi akan menampilkan skor yang diperoleh beserta total jumlah soal. Skor dihitung berdasarkan jumlah jawaban yang benar. Pengguna juga dapat mengulangi kuis dari awal apabila diinginkan.

Adapun tujuan dari aplikasi ini adalah sebagai media latihan pemahaman materi teknologi secara umum dan dapat digunakan untuk edukasi, latihan, atau sebagai hiburan.
